Dividing the grid

To set dividers between grid sections, use the class .grid-divider.

You won't always need a grid…

For a full-width layout, a grid isn't required.

Use .outer-block to set a max-width of 1020px and within it .inner-block to set left and right padding.

In this page, .outer-block is set on main and .inner-block set on .content.

Find these mixins in the GOV.UK frontend toolkit grid_layout.scss file.

Typography Example: Details summary

This example page demonstrates conditionally revealing information, using the HTML5 details and summary elements, they are described here.

These elements are only supported by a few modern browsers at the moment so you'll need a JavaScript polyfill to make them work in other browsers. Here's the polyfill used by GOV.UK elements.

The polyfill uses the aria-controls attribute on the controlling element to associate it with the section it shows/hides. This is combined with the aria-expanded attribute on the section to inform the accessibility API of the changes to the document.
